# Linux System Administration Course Repository

Welcome to my personal, production-grade Linux Systems Administration repository! This workspace serves as an organized, high-quality reference index of deployment plays, system hardening rulesets, operational analytics setups, and networking configurations.

---

## 📂 Repository Layout Map

* 📁 [01 - User & Access Management](01-user-and-access-management/README.md)
  * `1.1-users-and-groups.md` — Profile provisioning and system aging policies.
  * `1.2-file-permissions-and-umask.md` — Standard permissions, masking, and high-order bits.
  * `1.3-access-control-lists-acl.md` — Fine-grained access maps and backup schedules.
* 📁 [02 - Networking & Security](02-networking-and-security/README.md)
  * `2.1-network-configuration.md` — Network interface mapping, diagnostics, and routing.
  * `2.2-ssh-server-hardening.md` — Hardening configurations and asymmetric key handshakes.
  * `2.3-firewall-ufw.md` — Firewall wrapper operations, defaults, and rules.
* 📁 [03 - System Administration](03-system-administration/README.md)
  * `3.1-package-management-and-services.md` — Package tools, systemd managers, and logs.
  * `3.2-storage-management.md` — Partition schemes, file system formatting, and mounting.
  * `3.3-cron-jobs.md` — Job scheduling syntax, access tables, and scripting.
* 📁 [04 - Monitoring & Troubleshooting](04-monitoring-and-troubleshooting/README.md)
  * `4.1-log-analysis.md` — System logger structures and real-time security parses.
  * `4.2-system-metrics.md` — Processor tracking, storage activity, and load metrics.
  * `4.3-network-troubleshooting.md` — Socket diagnostics, nameserver checks, and link testing.
* 📁 [05 - Web Stack Deployments](05-web-stack-deployments/README.md)
  * `5.1-lamp-lemp-stack.md` — Persistent Apache, Nginx, and SQL structures.
  * `5.2-ssl-certification.md` — SSL/TLS setup via Certbot and protocol hardening.
  * `5.3-web-app-deployment.md` — Dynamic Web Application workspace hardening.
