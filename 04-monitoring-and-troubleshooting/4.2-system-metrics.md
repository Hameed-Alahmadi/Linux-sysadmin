## 🧠 Processor and RAM Diagnostics

Inspect active scheduler queues, process runtimes, core performance, and system memory maps.

```bash
# Real-time task manager displaying CPU registers and system threads
top

# Check system memory status (human-readable metrics)
free -h

# Check detailed active virtual memory statistics and queue states at 1-second increments
vmstat 1

# Display historical CPU, memory, and device stats (requires sysstat package)
sar -u 1 5
```

---

## 💾 Hard Disk and I/O Activity Metrics

Analyze performance limitations, raw block throughput, and active execution bottlenecks on hard storage disks.

```bash
# Real-time report of storage block reading and writing activity
iostat -xz 1 5

# Stream live input-output read and write operational metrics per system process
sudo iotop
```

---

## ⚙️ Process Auditing and Resource Constraints

Identify rogue processes, check execution priority ranks, and modify scheduling weights.

```bash
# Snapshot all running system processes mapped in a clean tree configuration
ps auxf

# List process threads, resource ownerships, and unique PIDs for a specific binary
pgrep -l nginx

# Elevate or lower running process prioritization weights (Nice Value: -20 to 19)
# lower value (-20) equals maximum priority, standard default is 0
sudo renice -n -5 -p 12345
```
