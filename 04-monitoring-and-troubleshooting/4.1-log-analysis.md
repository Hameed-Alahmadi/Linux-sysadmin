## 📂 Core Log Storage Architecture

On modern Linux environments, log collections are handled by `rsyslog` (traditional flat-file formatting) and `systemd-journald` (binary structured logging).

```bash
# General template inside /etc/rsyslog.conf mapping services to log files
# auth,authpriv.* /var/log/auth.log
# *.*;auth,authpriv.none          -/var/log/syslog
# cron.* /var/log/cron.log
# mail.* -/var/log/mail.log

# Primary local system log path locations:
# /var/log/syslog        - Global system activity log (Debian/Ubuntu)
# /var/log/messages      - Global system activity log (RHEL/CentOS)
# /var/log/auth.log      - Security and authentication registries (Debian/Ubuntu)
# /var/log/secure        - Security and authentication registries (RHEL/CentOS)
```

## 🔍 Parsing and Analyzing Flat-File Logs

Interrogate, stream, and isolate actionable anomalies from standard ASCII system logs.

```bash
# Real-time tracking of log files (streams additions live to your screen)
tail -f /var/log/syslog

# View the last 50 entries of the security access logs
tail -n 50 /var/log/auth.log

# Search for failed authentication events inside the security audit registers
grep -i "failed" /var/log/auth.log

# Count the total number of unique invalid ssh login attempts by IP address
grep "Invalid user" /var/log/auth.log | awk '{print $10}' | sort | uniq -c | sort -nr
```

## 🔄 Automated Log Rotation Control (`logrotate`)

Logrotate manages the automatic rotation, compression, and removal of historical log assets.

```bash
# Default config template configuration: /etc/logrotate.d/nginx
# /var/log/nginx/*.log {
#     daily            # Rotate logs every single day
#     missingok        # Do not throw errors if log file is empty/missing
#     rotate 14        # Retain exactly 14 history iterations before purging
#     compress         # Compress rotated files using gzip (.gz)
#     delaycompress    # Postpone compression until the next rotation cycle
#     notifempty       # Do not rotate logs if they are completely empty
#     create 0640 nginx adm  # Re-create the empty log file with permissions and owners
#     sharedscripts    # Execute the postrotate script only once for all matched logs
#     postrotate
#         systemctl reload nginx > /dev/null 2>&1
#     endscript
# }

# Force a manual diagnostic execution of logrotate configuration rules
sudo logrotate -f /etc/logrotate.conf
```
